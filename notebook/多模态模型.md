# 启动服务

```bash
vllm serve /root/home/my_model/Qwen2-VL-7B-Instruct \
	--served-model-name Qwen2-VL-7B-Instruct \
	--allowed-local-media-path / \
	--max-model-len 16384 \
	--trust-remote-code \
	--limit-mm-per-prompt image=4 \
	--mm-processor-kwargs '{"min_pixels": 784, "max_pixels": 1003520}'
```
